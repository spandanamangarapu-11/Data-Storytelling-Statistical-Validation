# Data Storytelling & Statistical Validation

## Task - 4 (ApexPlanet Internship)

### Objective

To transform data analysis findings into a compelling business story and validate key insights using statistical methods.

## Technologies Used

* Python
* Pandas
* Matplotlib
* SciPy

## Dataset

* Sample Superstore Dataset

## Project Overview

This project analyzes retail sales data to identify business trends and generate actionable insights. Data visualizations are used to communicate findings effectively, while hypothesis testing is performed to statistically validate the impact of discounts on profitability.

## Key Analysis

* Sales by Category
* Profit by Category
* Sales Distribution by Region
* Top 10 States by Sales
* Hypothesis Testing using Independent T-Test

## Hypothesis

**Null Hypothesis (H₀):** Discounts do not significantly affect profit.

**Alternative Hypothesis (H₁):** Discounts significantly affect profit.

## Results

The T-Test was performed to compare profits from discounted and non-discounted products. The p-value was used to determine whether discounts have a statistically significant impact on profitability.

## Business Insights

* Technology is the highest-performing sales category.
* Profitability varies across product categories.
* Regional sales performance differs significantly.
* Discount strategies should be optimized to maximize profit.

## Conclusion

This project demonstrates how data storytelling and statistical validation can support data-driven business decisions by transforming raw data into meaningful insights.
